What actually happened in 2016?

The 2016 Presidential election was momentous, but too many people disagree about what transpired. If you misdiagnose the patient, you treat the wrong disease and they don't get better. We need to know what happened so we can handle the future, and maybe change it.

Some people primarily blame Bernie voters, some blame minority voters, some blame racist Trump voters, but they can't all be right at the same time; so what was it?
